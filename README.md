# Frida Extract Keystore

Automatically extract KeyStore objects and relative password from Android applications with Frida - Read more: [ceres-c.it](https://ceres-c.it/2018/12/16/frida-android-keystore/)

Forked from https://gist.github.com/ceres-c/cb3b69e53713d5ad9cf6aac9b8e895d2

## Usage

```
npm install
python frida-extract-keystore.py <app_name>
```
